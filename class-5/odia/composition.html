Composition — Essay & Application Writing

ଓଡ଼ିଆରେ ପ୍ରବନ୍ଧ, ଆବେଦନ ପତ୍ର ଓ ଚିଠି ଲେଖିବା ଶିଖ। ବାକ୍ୟ ଗଠନ ଓ ଅନୁଚ୍ଛେଦ ଲେଖନ ଅଭ୍ୟାସ କର।

✍️
ରଚନା ବିଷୟ — Composition Topics
📄

ପ୍ରବନ୍ଧ ଲେଖନ

ଗାଈ, ଆମ ବିଦ୍ୟାଳୟ, ସ୍ୱାଧୀନତା ଦିବସ ଆଦି ବିଷୟରେ ପ୍ରବନ୍ଧ ଲେଖିବା ଶିଖ।

ପ୍ରବନ୍ଧ: ଆମ ବିଦ୍ୟାଳୟ
ଭୂମିକା → ବର୍ଣ୍ଣନା → ଶିକ୍ଷକ → ବନ୍ଧୁ → ଉପସଂହାର
📬

ଆବେଦନ ପତ୍ର

ପ୍ରଧାନ ଶିକ୍ଷକଙ୍କ ନିକଟରେ ଛୁଟି ପାଇଁ ଆବେଦନ ପତ୍ର ଲେଖିବା ଶିଖ।

ଠିକଣା → ତାରିଖ → ସେବାରେ → ବିଷୟ → ମୂଳ ଅଂଶ → ଧନ୍ୟବାଦ → ନାମ
💌

ଚିଠି ଲେଖନ

ବନ୍ଧୁ ବା ସମ୍ପର୍କୀୟଙ୍କୁ ଚିଠି ଲେଖିବା ଶିଖ। ଅନୌପଚାରିକ ଚିଠିର ନିୟମ।

ପ୍ରିୟ ବନ୍ଧୁ...
ତୁମ ସ୍ୱାସ୍ଥ୍ୟ ଭଲ ଥିବ ଆଶା କରୁଛି...
📝

ଅନୁଚ୍ଛେଦ ଲେଖନ

ଗୋଟିଏ ବିଷୟରେ ୫-୬ ବାକ୍ୟରେ ଅନୁଚ୍ଛେଦ ଲେଖିବା ଅଭ୍ୟାସ କର।

ନିୟମ: ମୁଖ୍ୟ ବାକ୍ୟ → ବିସ୍ତାର → ଉଦାହରଣ → ସମାପ୍ତି
💡 ଭଲ ରଚନା ଲେଖିବାର ନିୟମ
  • ପ୍ରଥମେ ବିଷୟ ବାରେ ଭାବ — ଯାହା ଲେଖିବ ତାହାର ଏକ ପ୍ରାରୂପ ତିଆରି କର
  • ଭୂମିକା, ମୂଳ ଅଂଶ ଓ ଉପସଂହାର ରଖ
  • ସରଳ ଓ ସ୍ପଷ୍ଟ ବାକ୍ୟ ଲେଖ
  • ବାନାନ ଓ ବିରାମ ଚିହ୍ନ ଠିକ୍ ରଖ

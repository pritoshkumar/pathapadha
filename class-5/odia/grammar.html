📝 ବ୍ୟାକରଣ

ବ୍ୟାକରଣ

Grammar — Sandhi, Samasa & Pratyaya

ଓଡ଼ିଆ ବ୍ୟାକରଣର ସନ୍ଧି, ସମାସ ଓ ପ୍ରତ୍ୟୟ ଶିଖ। ଶବ୍ଦ ଗଠନ ଓ ବାକ୍ୟ ରଚନା ଅଭ୍ୟାସ କର।

📝
ବ୍ୟାକରଣ ବିଷୟ — Grammar Topics
🔗

ସନ୍ଧି

ଦୁଇ ଶବ୍ଦ ଯୋଡ଼ି ନୂଆ ଶବ୍ଦ ତିଆରି ହୁଏ। ସ୍ୱରସନ୍ଧି ଓ ବ୍ୟଞ୍ଜନସନ୍ଧି ଶିଖ।

ହିମ + ଆଳୟ = ହିମାଳୟ
ମହା + ଈଶ୍ୱର = ମହେଶ୍ୱର
ବିଦ୍ୟା + ଆଳୟ = ବିଦ୍ୟାଳୟ
🧩

ସମାସ

ଦୁଇ ବା ଅଧିକ ଶବ୍ଦ ମିଳି ଗୋଟିଏ ନୂଆ ଶବ୍ଦ ହୁଏ। ତତ୍ପୁରୁଷ, ଦ୍ୱନ୍ଦ୍ୱ, ବହୁବ୍ରୀହି ସମାସ ଶିଖ।

ରାଜପୁତ୍ର = ରାଜାର ପୁତ୍ର
ପାଠଶାଳା = ପାଠ ପଢ଼ିବା ଶାଳା
ନୀଳାମ୍ବର = ନୀଳ ଅମ୍ବର ଯାହାର
📌

ପ୍ରତ୍ୟୟ

ଶବ୍ଦ ଶେଷରେ ଯୋଡ଼ାଯାଉଥିବା ଅଂଶ। କୃତ୍ ପ୍ରତ୍ୟୟ ଓ ତଦ୍ଧିତ ପ୍ରତ୍ୟୟ ଶିଖ।

ଚାଲ + ଇବା = ଚାଲିବା
ଲେଖ + ଆ = ଲେଖା
ପଢ଼ + ଅନ୍ତି = ପଢ଼ନ୍ତି
📋

ବିଭକ୍ତି

ଶବ୍ଦର ଶେଷରେ ଯୋଡ଼ାଯାଉଥିବା ଚିହ୍ନ। କର୍ତା, କର୍ମ, ସମ୍ବନ୍ଧ ବିଭକ୍ତି ଶିଖ।

ରାମ = କର୍ତା (ଶୂନ୍ୟ ବିଭକ୍ତି)
ରାମକୁ = କର୍ମ ବିଭକ୍ତି
ରାମର = ସମ୍ବନ୍ଧ ବିଭକ୍ତି
💡 ବ୍ୟାକରଣ ଶିଖିବାର ନିୟମ
  • ସନ୍ଧି = ଦୁଇ ସ୍ୱର/ବ୍ୟଞ୍ଜନ ମିଳନ
  • ସମାସ = ଅନେକ ଶବ୍ଦ ମିଳି ଏକ ଶବ୍ଦ
  • ପ୍ରତ୍ୟୟ = ଶବ୍ଦ ଶେଷରେ ଯୋଡ଼ାଯାଏ
  • ବିଭକ୍ତି = ବାକ୍ୟରେ ଶବ୍ଦର ସମ୍ପର୍କ ଦେଖାଏ

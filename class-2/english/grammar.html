📝 Basic Grammar

ଇଂରାଜୀ ବ୍ୟାକରଣର ସରଳ ନିୟମ ଶିଖନ୍ତୁ — Learn simple English grammar rules with examples!

📌 Using is, am, are
When to use is, am and are in sentences
📘 The Rule
I → use am
He / She / It (one person or thing) → use is
We / You / They (more than one) → use are
ମୁଁ → am ବ୍ୟବହାର କରନ୍ତୁ
ସେ (ପୁରୁଷ/ସ୍ତ୍ରୀ/ବସ୍ତୁ) → is ବ୍ୟବହାର କରନ୍ତୁ
ଆମେ / ତୁମେ / ସେମାନେ → are ବ୍ୟବହାର କରନ୍ତୁ
SubjectVerbExample Sentence
IamI am a student.
HeisHe is my friend.
SheisShe is a teacher.
ItisIt is a cat.
WeareWe are happy.
YouareYou are tall.
TheyareThey are playing.
👉 Using this, that, these, those
Pointing to things near and far
📘 The Rule
This → one thing, near you 👈
That → one thing, far from you 👉
These → many things, near you 👈
Those → many things, far from you 👉
This = ଏହା (ନିକଟ, ଏକବଚନ)  |  That = ସେହା (ଦୂର, ଏକବଚନ)
These = ଏଗୁଡ଼ିକ (ନିକଟ, ବହୁବଚନ)  |  Those = ସେଗୁଡ଼ିକ (ଦୂର, ବହୁବଚନ)
WordNear/FarExample
ThisNear (one)This is my book. 📖
ThatFar (one)That is a big tree. 🌳
TheseNear (many)These are my pencils. ✏️
ThoseFar (many)Those are birds. 🐦
📎 Using a and an
Articles — when to use a and when to use an
📘 The Rule
Use a before words that start with a consonant sound (b, c, d, f, g...)
Use an before words that start with a vowel sound (a, e, i, o, u)
ବ୍ୟଞ୍ଜନ ସ୍ୱରରେ ଆରମ୍ଭ ହେଉଥିବା ଶବ୍ଦ ପୂର୍ବରେ a ବ୍ୟବହାର କରନ୍ତୁ
ସ୍ୱର ସ୍ୱରରେ ଆରମ୍ଭ ହେଉଥିବା ଶବ୍ଦ ପୂର୍ବରେ an ବ୍ୟବହାର କରନ୍ତୁ
ArticleWhen?Examples
aConsonant sounda cat, a dog, a book, a mango
anVowel soundan apple, an egg, an umbrella, an orange
⏰ Simple Present Tense
Talking about things that happen regularly
📘 The Rule
Use the simple present tense for things that happen every day or are always true.
Add -s to the verb when the subject is he, she, it.
ପ୍ରତିଦିନ ଘଟୁଥିବା ବା ସର୍ବଦା ସତ୍ୟ ଥିବା କଥା ପାଇଁ ସାଧାରଣ ବର୍ତ୍ତମାନ କାଳ ବ୍ୟବହାର କରନ୍ତୁ।
he, she, it ପରେ କ୍ରିୟାପଦରେ -s ଯୋଡ଼ନ୍ତୁ।
SubjectVerbExample
IplayI play every day.
HeplaysHe plays cricket.
ShereadsShe reads a book.
WeeatWe eat rice.
TheyrunThey run fast.
ItrainsIt rains in July.

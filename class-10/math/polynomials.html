ବହୁପଦୀ — Polynomials

📊 ବହୁପଦୀର ଶୂନ୍ୟ (Zeros of Polynomial)

p(x) ର ଶୂନ୍ୟ ହେଉଛି ସେହି x ମୂଲ୍ୟ ଯେଉଁଠି p(x) = 0 ହୁଏ।

p(x) = x² - 5x + 6 ⟹ x = 2 ବା x = 3
  • ରେଖିକ ବହୁପଦୀର ୧ ଟି ଶୂନ୍ୟ
  • ବର୍ଗ ବହୁପଦୀର ସର୍ବାଧିକ ୨ ଟି ଶୂନ୍ୟ
  • ଘନ ବହୁପଦୀର ସର୍ବାଧିକ ୩ ଟି ଶୂନ୍ୟ

ଶୂନ୍ୟ ଓ ସହଗଙ୍କ ସମ୍ପର୍କ (Relationship)

ax² + bx + c ର ଶୂନ୍ୟ α ଓ β ହେଲେ:

α + β = -b/a
α × β = c/a
  • ଶୂନ୍ୟ ଯୋଗ = -b/a
  • ଶୂନ୍ୟ ଗୁଣଫଳ = c/a
  • ଏହି ସମ୍ପର୍କ ବ୍ୟବହାର କରି ବହୁପଦୀ ଗଠନ

ବିଭାଜନ ଆଲଗୋରିଦମ (Division Algorithm)

p(x) = g(x) × q(x) + r(x), ଯେଉଁଠି deg(r) < deg(g)

x³ - 3x² + 3x - 2 ÷ (x-2) = x² - x + 1, ଶେଷ = 0
  • ଦୀର୍ଘ ବିଭାଜନ ପଦ୍ଧତି
  • ଗୁଣନୀୟକ ନିର୍ଣ୍ଣୟ
  • ବାକି ପ୍ରମେୟ ପ୍ରୟୋଗ
